reachable-size

(reachable-size node)

Calculates the total size of data reachable from the given node. Expects a node map with ::size and ::links entries.

Raw blocks and nodes with no links have a total size equal to their ::size. Each link in the node’s link table adds its ::link/rsize to the total. Returns nil if no node is given.

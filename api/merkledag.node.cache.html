node-cache

(node-cache base & {:as opts})

Create a new empty cache which will store nodes in memory up to the given total size limit given. Note that nodes donâ€™t actually take up the same amount of memory as their backing blocks, but it is a useful proxy for their data size.

Note: this constructor does not perform an initial reaping, so it is possible to create a cache with more nodes stored than the limit permits.

Options may include:

  • :total-size-limit limit the total size of nodes stored in the cache
  • :node-size-limit if set, prevent the caching of nodes larger than this

format-block

(format-block codec value)

Serialize the given data value into a block using the codec. The value should be a node map, and must contain either a ::links table or a ::data entry.

Returns a block containing the formatted content and extra node attributes, or nil if value is nil.

parse-block

(parse-block codec block)

Attempt to parse the contents of a block with the given node codec. The codec should return a map of attributes to merge into the block; typically including ::links and ::data fields with the decoded node information.

Returns an updated version of the block with node keys set, or nil if block is nil.
